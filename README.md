# Макет сайта отчетности склада

Стартовый макет нового сайта:
- Визуально похож на предыдущий.
- Блоки: Исходящий поток и Входящий поток.
- Миниблоки: Сборка/сортировка, Упаковка, Приемка/раскладка, Инвентаризация.
- Каждая строка показателя кликабельна и ведет в аналитику.
- График с tooltip по точке: дата и значение.
- История 62 дня.
- Выбор периода 7–62 дня.
- Удаление значения.
- Очистка сегодняшнего дня и всей истории через модальное окно логин/пароль.
- Пока хранение в localStorage как макет. После утверждения полей можно подключить Supabase.
